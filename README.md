# 🚀 Quotes API + Frontend

Proyecto Full Stack compuesto por una API REST con **ASP.NET Core** y un frontend simple (generado con ayuda de IA) para consumirla.

## 🛠️ Tecnologías usadas

### Backend
* **ASP.NET Core**
* **Entity Framework Core**
* **AutoMapper**
* **C#**
* **SQL Server**

### Frontend
* **HTML, CSS, JavaScript**
* **Live Server (VS Code)**
* **Fetch API**

## 📌 Funcionalidades de la API

* ✅ Obtener todas las frases
* ✅ Filtrar por categoría
* ✅ Obtener frase aleatoria
* ✅ Buscar por ID
* ✅ Crear nuevas frases
* ✅ Eliminar frases

## ▶️ Cómo ejecutar el proyecto

### 🔧 Backend (ASP.NET Core)

1.  Abrir el proyecto con **Visual Studio**.
2.  Ejecutar el proyecto con perfil **HTTPS**.
3.  La API estará disponible en:  
    `https://localhost:7252` (o el puerto que indique Visual Studio).

### 🌐 Frontend

1.  Abrir la carpeta public en **VS Code**.
2.  Instalar la extensión **Live Server**.
3.  Click derecho en `index.html` → **Open with Live Server**.
